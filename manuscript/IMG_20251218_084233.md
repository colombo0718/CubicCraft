Code Craft 介紹

Code Craft 簡稱 CC，是一款自由度極高的太空模擬遊戲，玩家駕駛著各種機體來面對太空中可能遇到的所有挑戰。

此遊戲蘊含濃厚的物理、數學及程式的色彩。CC 宇宙中所有的物體都是由一顆顆等大的礦石球體所構築而成，機體的所有運動都是比照真實世界的方程式演算而來，而機體的控制不論是手動還是自動，背後都要藉由編寫 Blockly 的程式來達成。

在遊戲一開始，玩家所扮演的角色是一名 Leaf Lune 太空軍事學校的新兵，隨著故事的發展，解鎖不同的關卡、機體功能、以及遊戲內容，大致上而言，三個學年主要如下：

一、機體操縱與競速
二、外物閃避與射擊
三、外物碰撞與控制
