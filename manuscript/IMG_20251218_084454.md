Code Craft 色彩與光源

在遊戲場景中，礦石顆粒是以 MeshLambert Material 作為渲染材質。可對光源作出明暗效果，但不會反光是為霧面處理。顏色的設定上 RGB 三者皆為 44 ~ ff 解析度為 11，於 editor 中玩家可對機體進行塗裝。

光源的部分以目前規劃主要有三個，分別來自恆星、觀察者 (Viewer) 以及引擎 (engine)。三者有各自不同的渲染方法、作用範圍及意涵。

[圖：一個太陽圖示。]
[圖：一個半圓陰影的圓球圖示，代表受光面的星體。]

在漆黑的宇宙中，恆星是唯一的光芒，雖然遙遠而微弱，卻足以為人們帶來方向。遊戲中恆星位置設定在正 x 方向很遠的地方，因此光線是 DirectionalLight 來自正 x 強度 0.5。面向恆星的那一邊會顯現出顏色，而另一邊則為純黑色。從不同角度看就如同月亮的圓缺，於遊戲過程中看顏色就能辨明方向。在場景的正 x 方向也會有跟隨相機的恆星。

[圖：太陽圖示。圓球圖示（帶有受光面）。下方一個手電筒或探照燈圖示，標註為 Viewer。]

2016.06.06 (一)

若只有恆星的光，當鏡頭走到其背光處將看不到任何東西。為此將於相機上附帶光源，為模仿真實世界探照燈的效果，使用 SpotLight。自光距離 500 目標鎖定 craft。

[圖：太陽圖示。圓球圖示，後方有標註 fire 的噴射尾跡。下方探照燈圖示。]

當機體的引擎啟動後，除了產生火炎的線條外，還附帶有火光。是為圓心落在火炎前端的 Point Light，半徑只有 2。顏色為機體的固有色 (Natural Color)。此顏色套用在火光、炎末以及尾跡 (tail line)。越長的尾跡代表機體的速度越快。

[圖：一個帶有尾跡的機體草圖。]

上述的幾項光源，在未來第二學年之後還會有更多變化與應用。機體開始置入駕駛艙後不能使用觀察者視視角，燈光更另外加裝，兩機對戰的戰場可能是沒有恆星「夜戰」，此時互相不知對方的位 [?] 除了被光照到、啟動引擎及開射擊的瞬間。
